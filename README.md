# boatrace-forecast 🚤

ボートレース（競艇）の**展開予想**Webアプリ。出走表を入力すると、スタートから
1マーク（第1ターンマーク）までの攻防を予測し、**Canvasアニメーションで可視化**します。

## 特徴

- **展開予想ビジュアライザー**: スタート〜1マークの攻防をアニメーションで再生。
  予想1着艇が内を小回りして先マイ、下位艇は外を大回り…といった航跡を描画します。
- **決まり手予測**: 逃げ / 差し / まくり / まくり差し を自動判定。
- **着順予想と勝率**: 各艇の勝率をバー表示。
- **推奨買い目**: 3連単・2連単・3連複の買い目を信頼度付きで提示。
- 依存ライブラリ・ビルド不要のバニラ HTML / CSS / JavaScript 構成。

## 使い方

ローカルで簡易HTTPサーバを立てて開きます（`file://` でも動作しますが推奨はサーバ起動）。

```bash
# リポジトリ直下で
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

1. 「サンプル読込」または「ランダム生成」で出走表を埋める（手入力も可）。
2. 各艇に **全国勝率 / モーター2連対率 / 平均ST** を入力。
3. **「展開を予想する」** を押すと、可視化・決まり手・着順・買い目が表示されます。
4. 「▶ 再生」でアニメーションをリプレイできます。

## 予想ロジック（概要）

各艇の「強さ」を以下の要素から算出します（`js/engine.js`）。

| 要素 | 内容 |
| --- | --- |
| コース有利度 | インコースほど1着率が高いという競艇の基本特性（1号艇 約55% など） |
| 全国勝率 | 選手の総合的な実力 |
| モーター2連対率 | 機力 |
| 平均ST | スタートの速さ。特に外艇では速いSTが「まくり」成立の鍵 |

これらから1マーク到達時の主導権スコアを求め、着順・勝率・決まり手を導きます。

## ファイル構成

```
index.html        画面レイアウト
styles.css        スタイル
js/data.js        艇番カラー・コース別成績・サンプルデータ
js/engine.js      展開予想エンジン（強さ・勝率・決まり手・買い目）
js/visualize.js   1マーク攻防のCanvasアニメーション
js/app.js         UI制御
```
