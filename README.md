# boatrace-forecast 🚤

ボートレース（競艇）の**展開予想**Webアプリ。出走表を入力すると、スタートから
1マーク（第1ターンマーク）までの攻防を予測し、**Canvasアニメーションで可視化**します。

**公開URL（GitHub Pages）: https://zero-one-matsumoto.github.io/boatrace-forecast/**

iPhone の Safari でも上記URLを開くだけでフル機能（JavaScript）で動作します。

## 特徴

- **公式データ取得**: 場（24場）とレース番号を選んで「公式データ取得」を押すと、
  本日開催レースの出走表（選手名・勝率・モーター2連率・平均ST）を自動入力します。
- **展開予想ビジュアライザー**: スタート〜1マークの攻防をアニメーションで再生。
  予想1着艇が内を小回りして先マイ、下位艇は外を大回り…といった航跡を描画します。
- **決まり手予測**: 逃げ / 差し / まくり / まくり差し を自動判定。
- **着順予想と勝率**: 各艇の勝率をバー表示。
- **推奨買い目**: 3連単・2連単・3連複の買い目を信頼度付きで提示。
- 依存ライブラリ・ビルド不要のバニラ HTML / CSS / JavaScript 構成。

## 使い方

ローカルで簡易HTTPサーバを立てて開きます（`file://` でも動作しますが推奨はサーバ起動）。

```bash
# リポジトリ直下で
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

### iPhone（iOS Safari）で直接開く

iOS Safari で `file://` から複数ファイルを開くとサブリソース読み込みに失敗するため、
CSS/JS を1つに同梱した**単一HTMLファイル**を用意しています。

```bash
node build-standalone.js   # dist/boatrace-forecast.html を生成（生成済みもコミット済み）
```

`dist/boatrace-forecast.html` を iPhone に転送（AirDrop / メール添付 / iCloud Drive 等で
「ファイル」アプリに保存）し、ファイルをタップ → 共有 →「Safariで開く」で、
サーバ不要・オフラインでそのまま動作します。

1. **公式データ取得**（おすすめ）: 場とレース番号を選び「公式データ取得」を押すと、
   本日のレースの出走表が自動入力されます。または「サンプル読込」「ランダム生成」、手入力でも可。
2. 各艇に **全国勝率 / モーター2連対率 / 平均ST** を入力（取得時は自動入力）。
3. **「展開を予想する」** を押すと、可視化・決まり手・着順・買い目が表示されます。
4. 「▶ 再生」でアニメーションをリプレイできます。

### 公式データ取得について

- データ出典: **[BoatraceOpenAPI](https://boatraceopenapi.github.io/)**
  （公式の番組表データを整形しGitHub Pagesで公開しているJSON）。
  CORS対応のため、サーバ不要・静的フロント／iOS単一HTMLからでも直接取得できます。
- 取得対象は **本日開催分**。開催が無い場・締切後などは取得できない場合があります。
- 公式サイト `boatrace.jp` を直接 fetch しないのは、CORS制限でブラウザからは
  取得不可なため。OpenAPIのJSONは公式由来データをCORS可で配信しています。

## 予想ロジック（概要）

各艇の「強さ」を以下の要素から算出します（`js/engine.js`）。

| 要素 | 内容 |
| --- | --- |
| コース有利度 | インコースほど1着率が高いという競艇の基本特性（1号艇 約55% など） |
| 全国勝率 | 選手の総合的な実力 |
| モーター2連対率 | 機力 |
| 平均ST | スタートの速さ。特に外艇では速いSTが「まくり」成立の鍵 |

これらから1マーク到達時の主導権スコアを求め、着順・勝率・決まり手を導きます。

## ファイル構成

```
index.html        画面レイアウト
styles.css        スタイル
js/data.js        艇番カラー・コース別成績・サンプルデータ
js/engine.js      展開予想エンジン（強さ・勝率・決まり手・買い目）
js/visualize.js   1マーク攻防のCanvasアニメーション
js/fetch.js       公式データ由来の出走表取得(BoatraceOpenAPI)
js/app.js         UI制御
build-standalone.js   単一HTMLファイル生成スクリプト（iOS Safari向け）
dist/boatrace-forecast.html  生成済みの単一ファイル（オフライン/file://対応）
.github/workflows/deploy-pages.yml  GitHub Pages 自動デプロイ
```

## デプロイ（GitHub Pages）

`main` への push をトリガーに GitHub Actions が分割構成のまま
（`index.html` を起点に `styles.css` / `js/*.js` を）配信します。
Pages はワークフロー内で自動有効化されます。
